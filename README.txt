README

Theory of Computing Assignment, 2014

Compile and run EpsilonNFAtoDFA.java. A JFileChooser lets you choose
a file which contains an epsilon NFA in Grail+ format. Epsilon
transitions are represented by ?. The NFA from question 1 is stored
in q1.txt and the NFA from question 2 is stored in q2.txt. Results 
for the assignment problems are stored in Results.txt.
